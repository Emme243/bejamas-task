# Bejamas task
You can find the final project at [https://emmanuel-bejamas-task.netlify.app/](https://emmanuel-bejamas-task.netlify.app/) 

## Frontend Stack
- [TypeScript](https://www.typescriptlang.org/)
- [React](https://reactjs.org/) + [Next.js](https://nextjs.org/)
- [Redux Toolkit](https://redux-toolkit.js.org/)
- [TailwindCSS](https://tailwindcss.com/)
- [Graphql Apollo Client](https://www.apollographql.com/)
- Deployed on [Netlify](https://www.netlify.com/)

## Backend Stack
- [Node.js](https://nodejs.org/) + [Express.js](https://expressjs.com/)
- [Graphql Apollo Server](https://www.apollographql.com/)
- Deployed on [Heroku](https://www.heroku.com/)

## Database
- [MongoDB](https://www.mongodb.com/)
- [Mongoose](https://mongoosejs.com/)
- [MongoDB Atlas](https://www.mongodb.com/cloud/atlas/)
